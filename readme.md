# dataframe2ndjson
dataframe2ndjson converts a dataframe to ndjson file.

## Installation
```pip install dataframe2ndjson```

## How to use it?
```import dataframe2ndjson```
 ```dataframe2ndjson.df_to_ndjson(df,path)```
